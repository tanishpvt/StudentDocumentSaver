# StudentDocumentSaver

1 Data is Important in day to day life.<br/>
2 So i Developed the app named as Student Document Saver.<br/>
3 Student Can save all there documenets like marksheets,admission fee slip etc.<br/>
4 Student has a profile and document section.<br/>
5 where the document section contains all the data with CRUD operations.<br/>
6 And the profile section contains of the students details.
7 All the data are secured in DataBase(Firebase).<br/>
8 Encryption Used MD5 . SHA1, SHA256.<br/>
9 There is also admin panel to view how many users has registered.<br/>
10 The admin panel has OTP verification and the student has email verification.
